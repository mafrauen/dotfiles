## Install node.js

```sh
brew update && brew install node
```

## Install grunt

```sh
npm install -g grunt-cli
```

## Download dotfiles

```sh
git clone https://mafrauen@bitbucket.org/mafrauen/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
```

## Install

```sh
npm install && grunt
```
